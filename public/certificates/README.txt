CERTIFICATE FILES FOLDER
========================

Place your certificate PDF files in this folder using these exact filenames:

  hackathon-2025.pdf          -> Thoothukudi District Police Hackathon 2025
  ai-ml-certificate.pdf       -> AI Powered Machine Learning & Deep Learning
  web-app-testing.pdf         -> Web Application Testing
  nptel-elite-silver.pdf      -> NPTEL Elite + Silver (Industry 4.0)

The "View" button on each certification card will open the corresponding PDF
in an inline viewer modal. Files can be PDF or image format (PNG/JPG).

Note: If you have image certificates instead of PDFs, rename them with .pdf
extension won't work - update the `file` field in Certifications.tsx to use
the correct extension (e.g., .jpg or .png).
